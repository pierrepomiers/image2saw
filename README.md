# image2saw
Transforme une image en texture sonore séquencée et, en option, en vidéo vibrante synchronisée.  Chaque pixel devient un oscillateur (saw / sine / triangle / square) dont la fréquence est déterminée par sa luminosité.   La vidéo montre l’image “vibrer” autour d’une fenêtre glissante, avec une gaussienne centrée sur la zone en cours de lecture.
